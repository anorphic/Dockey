# Docker Container Monitoring Tool

A GUI-based monitoring tool for Docker containers built using **Python and PyQt6**.  
The application provides real-time insights into container performance, logs, and network activity through interactive visualizations.

## Features
- Monitor Docker container **CPU and memory usage**
- View **live container logs**
- Analyze **container network activity**
- Generate **network graphs and heatmaps**
- Simple and interactive **GUI dashboard**

## Technologies Used
- Python  
- PyQt6  
- Docker  
- Pandas  
- Matplotlib  
- Seaborn  
- NetworkX  
- Nmap  
- Trivy  

## Requirements
- Python 3.10+
- Docker installed and running
