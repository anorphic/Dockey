# Docker Container Monitoring Tool

The Docker Container Monitoring Tool is a GUI-based application developed using Python and PyQt6 that helps users monitor and analyze Docker containers in real time. The application provides a simple dashboard where users can observe container performance, inspect logs, and analyze network activity generated by containers. The goal of this project is to make container monitoring easier by presenting technical data in an interactive and visual format.

The tool allows users to monitor container CPU and memory usage, view live logs of running containers, and observe container status directly from the graphical interface. It also includes network monitoring capabilities that capture container network activity and identify communicating IP addresses. To make analysis easier, the application generates network graphs and heatmaps that visually represent traffic patterns and communication between containers and external systems.

The project is built using Python with the PyQt6 framework for the graphical user interface. It uses libraries such as pandas for data processing, matplotlib and seaborn for data visualization, and networkx for generating network graphs. External tools like Docker, Nmap, and Trivy are integrated to collect container information, analyze network traffic, and perform vulnerability scanning.

To run the project, Python 3.10 or higher must be installed along with Docker. After cloning the repository, install the required dependencies and run the application as shown below.

## Clone the Repository

```bash
git clone https://github.com/anorphic/DCM-Main.git
cd DCM-Main
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python main.py
```

This project is designed as an educational and monitoring tool to help users understand Docker container behavior, visualize container activity, and simplify container management through an easy-to-use graphical interface.
